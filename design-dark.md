# YNR Brand Design System: The "Pro" AI Labs Aesthetic (Dark Mode)

## 1. Design Atmosphere & Surface
*   **The Floor (Background):** `#0F0F10` (Deep Ink). This provides a rich, sophisticated base for a dark mode "Pro" aesthetic.
*   **The Layering (Content Cards):** `#1C1C1E` (Onyx). 
*   **The Depth (Elevation):** "Museum Gallery" dark style. 
    *   Border: 1px solid `rgba(255,255,255,0.08)`.
    *   Shadow: Soft ambient shadow `0 20px 50px rgba(0,0,0,0.4)`.
*   **Corner Radius:** 18px for primary cards to create a premium hardware feel.
*   **Atmospheric Detail:** A subtle radial spotlight gradient in the top-right background using `#409CFF` at 5% opacity.

## 2. Typographic Engine
*   **Headlines:** High-end Serif (e.g., Newsreader, Charter, or Source Serif Pro).
    *   **Weight:** 600 (Semi-bold).
    *   **Size:** 34px - 40px for Hero/Display.
    *   **Tracking:** -0.03em ("Tight" cadence).
    *   **Color:** `#F5F5F7` (Off-White Ink).
*   **UI & Data:** High-legibility Sans-Serif (e.g., Inter or Noto Sans).
    *   **Primary Body:** 17px (Editorial pace).
    *   **Secondary/Metadata:** 14px.
    *   **Color (Primary):** `#F5F5F7`.
    *   **Color (Secondary):** `#A1A1A6` (Neutral Gray).

## 3. Color Palette & Accents
*   **Primary Action:** Soft Electric Blue (`#409CFF`). A lighter, less "harsh" blue specifically for dark mode readability.
*   **Success:** `#32D74B`.
*   **Warning:** `#FF9F0A`.
*   **Error:** `#FF453A`.
*   **Neutral/Hairlines:** `#2C2C2E`.

## 4. Component Standards
*   **Primary CTAs:** Full Pill shape (9999px). Background: `#409CFF`, Text: `#FFFFFF`.
*   **Navigation Sidebar:** "Frosted Obsidian" effect. 
    *   Background: `rgba(28, 28, 30, 0.7)`.
    *   Backdrop-filter: `blur(25px)`.
*   **Status Indicators:** "Ghost Pills."
    *   Example (In Progress): Background `rgba(64, 156, 255, 0.1)`, Text `#409CFF`.
*   **Data Queues/Tables:** 
    *   No vertical borders.
    *   Horizontal hairlines only (`#2C2C2E`).
    *   Row Height: 64px (Minimum) to allow data to breathe.

## 5. Visual Personality
The YNR brand in dark mode represents "Quiet Intelligence." It remains minimalist and academic but adds a layer of focus and depth suitable for late-night professional work. Every element is precisely placed to maintain high contrast and editorial elegance.
